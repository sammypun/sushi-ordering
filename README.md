# Project Description

**Conveyor Sushi Restaurant Simulation** is a digital ordering and management system designed to modernize the traditional conveyor belt sushi dining experience. The system serves three main user groups: customers, service staff, and restaurant management.

For customers, the system provides a tablet-based interface at each table, allowing them to browse the menu, place orders, track their plate count, and process payments. The menu is organized by categories: Nigiri, Gunkan/Roll, Noodles/Soup, etc., featuring different plate colors representing various price tiers, making it easy for customers to understand pricing.

For service staff, the system helps ease order management, table assignments, and payment processing. It tracks plate counts by color: red, silver, gold, black, and automatically calculates total bills, reducing human error and improving service efficiency.

For management, the system provides insights into ordering patterns, table turnover rates, and sales data. The integration of table management, ordering, and payment systems helps optimize restaurant operations and enhance the overall dining experience.

# Key features

- Digital menu with categorized items and color-coded pricing
- Real-time order tracking and plate counting
- Automated bill calculation
- Table and tablet management
- Integrated payment processing

The system aims to enhance the modern sushi dining experience while maintaining the aspects of conveyor belt sushi restaurants.
